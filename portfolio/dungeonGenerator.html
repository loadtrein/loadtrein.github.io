Dungeon generator

« Back to portfolio

Based on the dungeon generation work by Tiny Keep, this project is a tool that procedurally generates randomly shaped and realistic dungeons to be used in a 2D game.

The user is able to modify the algorithm to his needs by setting the values of different parameters such as the number of rooms, the area of the world covered by the dungeon or the way the rooms are connected to each other.

This are its main characteristics:

  • Rooms creation following a Normal Distribution
  • Separation behaviour based on Craig Reynold's work
  • Use of Delaunay Triangulation to link the different rooms
  • Use of Prim's algorithm to create the minimum spanning tree connecting all rooms
  • Randomly shaped and realistic procedurally generated dungeons
  • Developed in C++ and OpenGL
