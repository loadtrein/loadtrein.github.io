DeckTools Inspire

« Back to portfolio

Main features I developed:

  • Procedural Generation of Meshes
    • Dividers Board Generation using polygon clippers with arbitrary shapes and dimensions
    • Stairs, Footings & Piers Generation

  • Gameplay & Tools
    • In-app Highlight System that renders the outline of the objects being selected
    • Improvements made to the 2D Drawing tools
    • Implemented adding Joists and Beams feature, movement of Stairs and Board Segments rotation
    • Blueprint generator from 3D wireframe by using face-culling and hidden lines removal algorithms

  • UI
    • New UI and improvements in the performance of its elements
    • Tutorial that updates dynamically based on user interaction and triggered events
    • Generic Modal System that instantiates and configures modals at runtime by the use of Reflection
    • Event driven Dictionary System to provide internationalisation to the app
