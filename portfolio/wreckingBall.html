Wrecking Ball

« Back to portfolio

Wrecking ball is a 2D game designed using Box2D physics engine and SFML API for displaying the multimedia content associated to the Box2D data in the screen.

The aim of the game is to completely destroy a building by controlling a wrecking ball. The user will be able to move the hook and the arm of the crane that are linked to the ball, and so will be able to control its position and velocity. The building will be damaged gradually depending on the strength of the impacts.

A little game HUD shows the current non-destroyed percentage of the building.


This are its main characteristics:

  • Use of several kinds of joints: Prismatic, Revolute, Rope, etc.
  • Collision filtering to create particles effect
  • Manifolds management to control the strenght of the impacts
  • Uses SFML API for displaying multimedia content
  • Developed in C++
