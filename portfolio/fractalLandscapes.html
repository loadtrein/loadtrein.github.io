Fractal Landscapes

« Back to portfolio

Outdoor realistic terrain rendering in real-time is an important feature for a large class of games and simulation applications.

This project generates landscapes using a stochastic algorithm designed to produce fractal behaviours, which mimic the appearance of natural terrain by displaying self-similar patterns, a property found in real outdoor landscapes.

By the use of different hotkeys, the user is able to navigate through the scene and to regenerate the terrain with an specific width, length and roughness. In addition, several transformations functions such as smoothing, thermal erosion and perturbation can be applied to the shape of the terrain.


Its key features are:

  • Fractal Landscape generation with Diamond-Square Algorithm
  • Terrain transformation functions: Smoothing, Thermal Erosion and Perturbation
  • Terrain texturing based on height with blending between intervals
  • Waves simulation based on multiple sine functions
  • Light reflections and bump mapping
  • Skybox and realistic water reflection
  • Developed in C++ and OpenGL with custom Shaders
